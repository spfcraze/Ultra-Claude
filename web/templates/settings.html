Setup Guide

1

Install the Service

Click "Install Service" to register UltraClaude as a system service. This creates the necessary service files for your platform (systemd on Linux, launchd on macOS).

2

Start the Service

Once installed, click "Start" to begin running UltraClaude in the background. The service will automatically start on system boot.

3

Configure Projects

Set up your projects with GitHub webhooks to receive automatic issue notifications. Enable "Auto-start" on projects for fully automated processing.

Webhooks

UltraClaude can receive webhooks from GitHub to automatically process new issues and track PR status. Configure webhooks in each project's settings after setting up the endpoint in GitHub.

Webhook Endpoint

http://your-server:8420/webhooks/github

Add this URL to your GitHub repository's webhook settings. Select events: Issues, Pull Requests.
