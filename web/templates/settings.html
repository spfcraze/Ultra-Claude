Webhooks

UltraClaude can receive webhooks from GitHub to automatically process new issues and track PR status. Configure webhooks in each project's settings after setting up the endpoint in GitHub.

Webhook Endpoint

http://your-server:8420/webhooks/github

Add this URL to your GitHub repository's webhook settings. Select events: Issues, Pull Requests.
